приложение для отчета по предмету Промышленное Программирование на Языке Java

для запуска потребуется Docker, зайти в корневую папку приложения, выполнить команду docker compose up

ui доступен на localhost:8080
backend доступен на порту localhost:8081

Приложение для создания и прохождения тестов

для авторизации использовать login=root password=root
